# 功能需求

1. 旅遊手冊app
2. 行動裝置為主的設計
3. 首頁為選擇哪次旅程
4. 旅程的內頁分別為三個大功能[行程表，店鋪，資訊]

    ## 行程表

    1. 項目顯示的資訊(簡易)
       - 標題
       - 地點
       - 類別
       - 預計到達/離開時間
       - 行程間的時間差顯示
       - 項目縮圖參照類別顯示[餐飲，住宿，景點]...等等
      
    2. 每個項目可點擊並從下向上彈出顯示詳細資訊(詳細)
        - 繼承簡易的資訊
        - 地點如有座標，則外連Google Map    
        

    ## 店鋪

    1. 項目可以有多個tag
    2. 以tag分類商店
    3. 名稱
    4. 地點
    5. 營業時間
    6. 可外連Google Map

    ## 資訊

    1. 能讀取markdown格式的文件
